# pemrograman-web-lanjut

Repository tiruan untuk mata kuliah Pemrograman Web Lanjut.

Struktur folder sama seperti contoh di GitHub:

```
app/
    Controllers/
    Core/
config/
public/
.htaccess
```

### Menjalankan di XAMPP
1. Letakkan folder `Pemograman Web Lanjut` di `C:\xampp\htdocs\`.
2. Jalankan `Apache` dari XAMPP Control Panel.
3. Pastikan `mod_rewrite` aktif.
4. Akses `http://localhost/Pemograman%20Web%20Lanjut/` di browser.

Aplikasi ini menggunakan satu controller utama dan rewrite ke folder `public/`.
